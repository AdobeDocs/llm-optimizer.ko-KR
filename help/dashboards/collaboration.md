# 공동 작업

공동 작업을 통해 팀원을 조직의 LLM Optimizer 작업 영역으로 초대하여 모든 사람이 한 곳에서 인사이트와 분석에 액세스할 수 있습니다. 동료를 추가하여 마케팅, SEO, PR 및 기술 팀이 브랜드에 맞는 가시성 개선을 위해 원활하게 협력하고, 결과를 공유하며, 기회에 대한 공동의 조치를 취할 수 있도록 합니다.

## 팀원을 초대하는 방법

**이메일 주소** 필드에 이메일을 추가하고 **초대 보내기**&#x200B;를 클릭하여 팀원을 초대할 수 있습니다. 수락하면 조직의 LLM Optimizer 작업 영역에 참여하게 됩니다.
