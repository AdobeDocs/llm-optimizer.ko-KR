# URL 관리자

URL 관리자를 사용하면 도메인의 특정 페이지가 AI 검색에서 어떻게 작동하는지 분석할 수 있습니다. URL 수준에서 가시성, 에이전틱 트래픽, 참조 데이터를 결합하여 인용되는 URL과 응답에 표시되는 빈도를 세부적으로 볼 수 있습니다.

![URL 관리자](/help/dashboards/assets/url-insp.png)

## 필터

페이지 맨 위에서 필터를 적용하여 보기를 세분화할 수 있습니다. 선택한 필터는 대시보드에 있는 **모든** 섹션에 영향을 미칩니다. 다음을 사용자 정의할 수 있습니다.

* **날짜 범위** - 표시된 데이터의 시간대를 선택합니다. 예를 들어 지난 4주를 선택합니다. **주 사용자 정의** 옵션을 선택하여 기간을 사용자 정의할 수도 있습니다.
* **카테고리** - 표시된 결과를 카테고리별로 필터링합니다.
* **플랫폼** - 분석할 AI 엔진을 선택합니다.
* **페이지 콘텐츠 유형** - 콘텐츠 유형별로 필터링합니다.
* **지역** - 지역별로 결과를 필터링합니다. 일부 지역은 시작 시 사용할 수 없습니다.

원하는 필터를 선택한 후 **필터 적용**&#x200B;을 클릭하여 대시보드에 선택 사항을 적용합니다.

## 개요 지표

URL 관리자는 AI 검색에서 페이지가 어떻게 작동하는지 빠르게 평가할 수 있도록 여러 개요 지표를 제공합니다. 제공되는 지표는 다음과 같습니다.

* **자체 소유 인용이 있는 고유 프롬프트** - 자체 소유 인용이 있는 고유 AI 프롬프트의 총 개수입니다.
* **총 고유 프롬프트 수**  - 고유 AI 프롬프트의 총 개수입니다.
* **고유 인용 URL** - 인용된 고유 소유 URL의 개수입니다.
* **총 인용 횟수** - AI 생성 응답에서 소유 URL이 인용된 총 횟수입니다.
* **총 에이전틱 히트 수**  - URL에 대한 AI 에이전트의 총 히트 수입니다.
* **LLM에서 발생한 참조 히트 수** - URL에 대한 AI 생성 응답에서 나온 총 히트 수입니다.

각 개요 지표에 대한 트렌드 지표는 이러한 값이 이전 기간과 비교하여 시간이 지남에 따라 어떻게 변화하고 있는지를 보여 줍니다.

## 인용된 URL

인용된 URL 보기는 AI 생성 응답에서 인용된 브랜드의 모든 URL을 지원 지표와 함께 나열합니다. 두 테이블 모두 주제에 빠르게 액세스할 수 있는 검색 필드가 있으며, **열 구성** 버튼을 클릭하여 표시되는 지표를 사용자 정의할 수 있습니다. 또한 **내보내기** 옵션을 사용하여 table .csv를 다운로드하고 팀과 인사이트를 공유하거나 이 테이블을 실행 보고서에 포함할 수 있습니다.

![인용된 URL](/help/dashboards/assets/cited-urls.png)

제공되는 지표는 다음과 같습니다.

* **URL** - 분석된 URL입니다.
* **인용된 횟수** - AI 생성 응답에서 URL이 인용된 횟수입니다.
* **다음 위치에 인용된 프롬프트** - URL을 인용한 고유한 AI 프롬프트 개수입니다.
* **카테고리** - URL과 관련된 제품 카테고리 또는 주제입니다.
* **지역** - URL이 인용된 지리적 지역입니다.
* **에이전틱 히트 수**  - URL에 대한 AI 에이전트의 총 히트 수입니다.
* **참조 히트 수** - URL에 대한 AI 생성 응답에서 전송된 히트 수입니다.

## 인용 경쟁 중인 트렌딩 URL

인용 경쟁 중인 트렌딩 URL 보기에서는 현재 브랜드와 관련된 응답에서 인용되고 있는 외부 URL을 강조 표시하여 해당 분야에서 누가 인용에서 우위를 점하고 있는지 측정합니다. 데이터 테이블에는 특정 URL에 빠르게 액세스할 수 있는 검색 필드가 있습니다. 또한 **내보내기** 옵션을 사용하여 table .csv를 다운로드하고 팀과 인사이트를 공유하거나 이 테이블을 실행 보고서에 포함할 수 있습니다.

![인용 경쟁 중인 트렌딩 URL](/help/dashboards/assets/trend-url.png)

제공되는 지표는 다음과 같습니다.

* **URL** - 분석된 URL입니다.
* **콘텐츠 유형** - 콘텐츠의 유형(소유, 소셜, 획득, 기타)입니다.
* **인용된 횟수** - AI 생성 응답에서 URL이 인용된 횟수입니다.
* **다음 위치에 인용된 프롬프트** - URL을 인용한 고유한 AI 프롬프트 개수입니다.
* **카테고리** - URL과 관련된 제품 카테고리 또는 주제입니다.
* **지역** - URL이 인용된 지리적 지역입니다.

### 세부 정보 창

인용된 보기와 트렌딩 보기 모두에서 URL은 각 행의 끝에 **세부 정보** 버튼이 있습니다. 버튼을 클릭하면 추가 세부 정보가 포함된 별도의 창이 표시됩니다. 세부 정보 창은 URL이 인용되는 빈도, <!--the sentiment of AI responses where it is mentioned,--> URL에 나타나는 주제와 프롬프트, 시간 경과에 따른 에이전틱 및 참조 트래픽의 트렌드를 보여 줍니다(소유 URL의 경우).
